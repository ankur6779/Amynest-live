Upgrade the Phonics Zone into a premium, highly interactive, age-adaptive learning system for kids aged 2–8.

GOAL:
Create a visually rich, engaging, and habit-forming phonics experience that delights both child and parent.

--------------------------------------------------
1. AGE-BASED LEARNING PATH (CORE)
--------------------------------------------------
Create 3 adaptive levels:

Age 2–4:
- Focus: Sound recognition
- Big visuals (Apple, Ball, Cat)
- Tap to hear sound
- No heavy text

Age 4–6:
- Focus: Letter + sound + simple words
- Phonics blending (c-a-t)
- Matching games

Age 6–8:
- Focus: Reading + spelling
- Digraphs (sh, ch, th)
- Word building + sentence level

--------------------------------------------------
2. DAILY PHONICS FLOW (STICKINESS ENGINE)
--------------------------------------------------
Section: “Today’s Learning for [Child Name]”

Daily flow:
1. Learn Sound (audio + animation)
2. Play Game (interactive)
3. Practice Words
4. Reward ⭐

Total time: 5–8 minutes

--------------------------------------------------
3. VISUAL-FIRST UI (VERY IMPORTANT)
--------------------------------------------------
- Large image for each letter (A → Apple 🍎)
- Animated highlight on letter
- Minimal text
- Bright kid-friendly UI

--------------------------------------------------
4. AUDIO SYSTEM
--------------------------------------------------
- Auto-play phonics sound
- Repeat button 🔊
- Friendly child voice

--------------------------------------------------
5. INTERACTIVE MINI-GAMES
--------------------------------------------------
Include:

1. Sound Match
   - Hear “B”
   - Tap correct image (Ball)

2. Tap the Letter
   - Hear sound → select letter

3. Word Builder
   - Drag letters → make word

4. Missing Letter
   - _at → cat

--------------------------------------------------
6. ANIMATION SYSTEM
--------------------------------------------------
- Letter pop animation
- Image bounce on tap
- Correct answer → confetti
- Wrong answer → gentle shake

--------------------------------------------------
7. PROGRESSION + REWARD
--------------------------------------------------
- Stars ⭐
- Daily streak 🔥
- Unlock next level
- “Great job!” screen

--------------------------------------------------
8. CONTENT EXPANSION
--------------------------------------------------
Include:

- A–Z phonics
- Short vowels (a, e, i, o, u)
- Long vowels (ai, ee, oa)
- Digraphs (ch, sh, th)
- Blends (bl, st, cr)

--------------------------------------------------
9. PARENT VIEW (WOW FACTOR)
--------------------------------------------------
Show:

- Today’s progress
- Sounds learned
- Suggested activity

Example:
"Your child learned 3 sounds today!"
