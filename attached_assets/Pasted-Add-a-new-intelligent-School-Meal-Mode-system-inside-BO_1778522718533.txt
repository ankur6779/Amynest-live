Add a new intelligent “School Meal Mode” system inside BOTH:

* Single Child Routine Generator
* Family Routine Generator

in mobile app and web app
====================================================
PRIMARY GOAL
============

Meal suggestions should NOT always generate automatically.

Instead:
before routine generation, ask parent:

“Do you need school snacks / packed lunch suggestions?”

Options:

* Yes
* No

If:
NO →
Skip school meal generation entirely.

If:
YES →
Generate intelligent school meal recommendations.

====================================================
NEW INPUT FIELD
===============

Add new routine generation input:

schoolMealMode

Values:

* disabled
* snack_only
* packed_lunch_only
* snack_and_packed_lunch

UI LABEL:
“Need school snacks or packed lunch suggestions?”

====================================================
APPLIES TO
==========

This logic must work across:

1. Single child routine generation
2. Family routine generation
3. Saved routine regeneration
4. AI regenerated routines
5. Adaptive re-generation

====================================================
CONDITIONAL GENERATION LOGIC
============================

IF schoolMealMode == disabled:

* completely skip school meal generation
* do not mention lunch/snacks
* do not consume AI tokens for meal generation
* remove meal sections from schedule

IF enabled:
Generate:

* age-appropriate meals
* allergy-safe meals
* travel-safe meals
* school-safe packed meals
* AQI/weather-aware meals
* region-aware meals
* temperature-aware meals
